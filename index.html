FAQ

Questions

Do I need n8n Cloud or can I self-host?
Both work. If you're self-hosting n8n (free), these workflows run at zero cost. If you're on n8n Cloud, they work exactly the same. The only running cost is your Google account (free) and the Claude API (only for workflow #2 — pay-per-use, pennies per proposal).
I'm not technical. Can I still use these?
Yes. Every workflow has a step-by-step sticky note inside n8n that tells you exactly what to configure. If you can connect a Google account and paste a spreadsheet ID, you're good. Setup takes 15 minutes total.
Can I modify the workflows for my agency?
Absolutely. The license covers personal and commercial use. Edit anything — email templates, branding, logic, triggers. They're fully open inside n8n.
What Google services does this use?
Gmail, Google Drive, and Google Sheets. All free. You connect once via OAuth in n8n and it applies to all workflows.
Does workflow #2 (AI Proposals) require a paid API?
It uses the Anthropic Claude API, which is pay-per-use. Generating one proposal costs roughly $0.01–$0.03 USD depending on length. If you send 50 proposals a month, that's about $1 in API costs.
